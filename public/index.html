Documentação da API

Capybara Controller

O CapybaraController é responsável por gerenciar as operações com as capivaras na API.

  • GET /capybara - Retorna todas as capivaras
  • GET /capybara/:id - Retorna uma capivara específica por ID
  • POST /capybara - Cria uma nova capivara
  • PATCH /capybara/:id - Atualiza uma capivara existente
  • DELETE /capybara/:id - Deleta uma capivara específica
